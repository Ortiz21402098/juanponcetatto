Tattoo Studio Manager - Solución de Gestión Integral
Este sistema es una plataforma web personalizada diseñada para optimizar la administración de un estudio de tatuajes. El objetivo central es centralizar la gestión de turnos, el seguimiento de clientes y el control financiero (ingresos y gastos) en una sola interfaz intuitiva.

Propósito del Proyecto
Desarrollado como una solución Freelance, este sistema resuelve la necesidad de digitalizar la agenda y las finanzas de un artista del tatuaje, permitiendo una visión clara de la rentabilidad del estudio y una organización eficiente de las sesiones.

Funcionalidades Clave
Gestión de Turnos: Calendario interactivo para la asignación, reprogramación y seguimiento de sesiones de tatuaje.

Control de Clientes: Base de datos con historial de trabajos, contacto y detalles específicos de cada sesión.

Módulo Financiero:

Registro de Ingresos: Seguimiento detallado de pagos por sesión.

Control de Gastos: Gestión de insumos (agujas, tintas, alquiler, etc.) para calcular la ganancia neta.

Reportes: Visualización rápida de ingresos mensuales y gastos acumulados.

Stack Tecnológico
Frontend:  HTML/ JS - Interfaz ágil y reactiva para una gestión rápida.

Estilos: Tailwind CSS / Bootstrap - Diseño moderno, oscuro y acorde a la estética del rubro.

Arquitectura del Sistema
El proyecto sigue un modelo de arquitectura limpia, separando las responsabilidades para facilitar el mantenimiento:

Capa de Datos: Modelado relacional para vincular cada gasto e ingreso a un turno o cliente específico.

Capa de Negocio: Lógica para el cálculo automático de saldos y balances financieros.

Capa de Presentación: Dashboard visual diseñado para su uso tanto en desktop como en tablets dentro del estudio.
